# Executar o Projeto

Clone o projeto

```bash
  git clone https://github.com/VagnerNerves/rn-reanimated-beta-version4.git
```

Entre no projeto

```bash
  cd rn-reanimated-beta-version4
```

Instale as dependências

```bash
  npm i
```

Gere o prebuild

```bash
  npx expo prebuild
```

Execute o projeto

```bash
  // Android
  npm run android

  // IOS
  npm run ios
```
